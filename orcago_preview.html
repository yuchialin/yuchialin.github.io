About the Project

As a product designer, I led the design of the new Digital Check-In feature for the mobile application Orca Go. Digital Check-in provides QR Code and GPS check-in which allows lecturers to easily keep track of the attendances and allows attendees to check-in virtually.

The Goals

Seamless Data Transfer

Minimize the data transfer work that the lecturers have to do

Smooth Experience

The lecturer and attendees will be able to accomplish check-in with ease

Stay Informed

Keep lecturer informed on attendance information

Design Process

Final Design

Track attendance of multiple events in one place

Frustration: Lecturers have to carry multiple attendance sheets around, so it is easy to use the wrong sheet.

Solution: Create an easy system for lecturers to track attendance of multiple events for different companies.

Real-time Attendance Information

Frustration: Lecturers can’t effectively track the attendance rates.

Solution: Enable lecturers to get real-time attendance rates through graph.

Hassle Free, GPS & QR Code Check-In Capability

Frustration: Paper and pen check-in significantly slows down the event schedule.

Solution: A feature that supports both GPS & QR Code check-in, so lecturers can start the event on time.

No More Attendance Sheet or Lost Data

Frustration: Attendance papers are easily misplaced, damaged, or otherwise not submitted to the hosting company, especially when events are off campus.

Goal: Enable lecturers to seamlessly log event attendance into the HR app without the need to transfer paperwork.
