About the Experience

For the first six months in Learning Digital, a tech firm that developed Learning Management System, I helped redesign some mini features for the platform and the mobile application.

The Challenges

Inconsistent Brand Identity

Designing a responsive web based platform and an application that fit the new brand identity.

Various Stakeholders

Designing a web platform and mobile application for various users, including human resource, managers, and employees.

Cross-Disciplinary

Communicating effectively with product manager, project manager and the developers to put design in practice.

Features

In addition to redesign the interfaces of each features, I also add and modify the functions of the features to make it more user-centric.

Rewards Point System

Goal: Find a clear way to inform how many reward points do users have.

Solution: A easier and salient display for users to look at.

Announcement

Goal: Ensure and track if the employees read the announcements.

Solution: A read button at the end of the announcement that allows users to confirm they read the announcement.

Mission Feature

Goals: Increase the flexibility of missions that the HR can create for the employees.

Solution: The HR can put different rules into one big mission.
